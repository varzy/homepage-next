CSS 有两种盒模型，目前大部分使用 IE 盒模型

- IE 盒模型：width = content + padding + border
- W3C 盒模型：width = content

切换盒模型的方式：


```scss
// W3C
box-sizing: content-box;
// IE
box-sizing: border-box;
```
