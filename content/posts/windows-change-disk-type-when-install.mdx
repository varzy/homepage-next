1. 在系统分区界面按 Shift+F10 打开命令提示符
2. 输入：`diskpart`
3. 输入：`list disk` ，打开磁盘信息，可以通过磁盘容量判断将要选择的磁盘
4. 输入：`select disk 0` ，0是指选择第0号磁盘
5. 输入：`clean` ，清空当前磁盘分区及数据
6. 输入：`convert mbr`，转换为 mbr 分区；输入：`convert gpt`，则转换为 gpt 分区
